## 🚀 Getting Started

### Installation (5 minutes)

```bash
cd mobile-app
npm install
npm install -g eas-cli
eas login  # Create free account
```

### Testing (15 minutes)

```bash
# Option 1: Android
npm run android

# Option 2: iOS (Mac only)
npm run ios

# Option 3: Web preview
npm run web
```

### Production Build (30 minutes)

```bash
npm run build:android
npm run build:ios
```

### App Store Submission (30 minutes)

```bash
npm run submit:android
npm run submit:ios
```
